最近通过一个视频教程快速复习一下HTML：

  • <header>标签通常包含一些介绍性或导航用的信息

  • <h1><h2>…<h8>等tag用于写标题，其中<h1>tag中的内容会被各种搜索引擎在计算关联度时使用，所以如果想让自己的网页被更多人检索到，你需要有一个<h1>tag，并且里面的内容很重要

  • 一个以前不知道的小知识点——Lorem Ipsum，也称乱数假文或者哑元文本， 是印刷及排版领域（也包括电子排版）所常用的虚拟文字。如今使用它的目的简单来说就是为了保持字母多多少少标准及平均的分配，而不是“此处有文本，此处有文本”，从而让内容更像可读的英语

  • id属性是唯一的，不能有多个元素的id相同，而class没有这个限制，即允许不同的元素具有相同的class。此外class还可以并列存在，例如class = "grouped newClass"

  • HTML本身与样式无关（极少数例外比如h1h2等等不同标题大小），它只与网页的结构和内容相关，样式全部交由CSS负责
